A steganography project that allows you to hide secret messages within image files for educational purposes.

## Getting Started

### Prerequisites

- Python 3.x
- Pillow library for image processing: `pip install Pillow`
- pip install stegano

### Installation

1. Clone the repository: `git clone https://github.com/yourusername/steganography-project.git`
2. Navigate to the project directory: `cd steganography-project`
3. Install dependencies: `pip install -r requirements.txt`

## Usage

1. Open the program and select the image you want to show/hide text.
2. Write the secret text on the right panel, then press the "Hide Text" button to store the text into the picture.
3. Press the "Save" button to save the new secret picture to a specified path.
4. To read the hidden text, reselect the new picture, and press the "Read" button. The secret text should appear on the right panel.
